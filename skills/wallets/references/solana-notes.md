# Solana Wallet Notes

## Summary
Solana wallet integration differs from EVM. Use Solana-specific tooling and RPC semantics.

## Guidance
- Use Solana JSON-RPC for account data and program interactions.
- Use DAS for NFT and compressed asset data.
