# Ecosystem — Adjacent Open-Source Tools

## Summary
Popular open-source libraries that pair well with Alchemy when building crypto apps. This section is intentionally concise.

## References (Recommended Order)

### EVM Libraries
1. [viem.md](references/viem.md) - Modern TypeScript Ethereum client with strong types.
2. [ethers.md](references/ethers.md) - Widely used Ethereum library for wallets and contracts.
3. [web3js.md](references/web3js.md) - Long-standing Ethereum JavaScript library.
4. [wagmi.md](references/wagmi.md) - React hooks for Ethereum with wallet connectors.
5. [rainbowkit.md](references/rainbowkit.md) - Wallet connection UI kit.

### EVM Development Frameworks
6. [hardhat.md](references/hardhat.md) - Ethereum development environment.
7. [foundry.md](references/foundry.md) - Rust-based Ethereum toolchain.

### Solana Libraries
8. [solana-web3js.md](references/solana-web3js.md) - Primary Solana JavaScript library.
9. [spl-token.md](references/spl-token.md) - Solana token program and management tools.
10. [anchor.md](references/anchor.md) - Solana framework for building programs.
11. [metaplex.md](references/metaplex.md) - Solana NFT standards and tooling.
